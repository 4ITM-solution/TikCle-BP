### v4에서 그대로 가져올 문법
1. **채널 = 1급 축, 항상 노출.** 상단 스트립의 채널별 카운트(TK 1.2K · Meta 179 · IG 450 · YT 250 · TT Shop $1.2M · Amazon), KPI 카드의 채널 분해 서브라벨("총 view 486M — YT 91% · IG 8%"), 섹션 안 채널별 뷰. 접어도 되는 건 측정치·기간·SKU 같은 보조 조합이지 채널이 아니다.
2. **시나리오 3종 전환**: 풀 데이터(SharkNinja류) / IG·YT만(Dyson류) / TK+Amazon+Meta(닥터리쥬올). 채널 구성이 다른 케이스에서 화면이 어떻게 달라지는지 목업에서 전환 가능하게. fixture는 rev1의 닥터리쥬올 실데이터 + 나머지 2종은 v4 수치 재사용 가능.
3. **데이터 채널 카드 그리드**(7개 중 N개 활성)와 **phase 진행 접기** 유지.
4. 정보 밀도는 v4 수준 — rev1처럼 화면을 비우지 말 것. 사용자는 밀도 높은 대시보드에 익숙하다.
