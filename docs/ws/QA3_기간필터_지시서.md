## 체크리스트 (화면·뷰·원천 3층)
1. **적용 동작**: 📆 행 커스텀에 2026-04-01~2026-07-12 입력 → 적용 → 보라 배너(적용 문구+범위 고지) 노출, 새로고침 후에도 유지(옵션 저장 확인: `cases.options.period_scope`)
2. **KPI 총 영상**: 기간 적용 후 TK 값이 기준값 3,522와 일치하는가 (±0 — distinct video id 기준이라 URL 중복이 있으면 오차 사유 기록)
3. **KPI 총 인플 풀**: TK 값 1,711 일치 여부 (phase2 캐시 override 확인 — **기간 값이 캐시 전 기간 값(7,858 스케일)보다 작아도 이겨야 함**. 크면 override 버그)
4. **A 차트**: 4월 이전 막대가 0으로 비고 4~7월만 값 있는가. 광고 비중 KPI ~72% 근처인가
5. **B Top 작성자**: 기간 밖에서만 활동한 인플이 명단에서 빠졌는가 (표본 2~3명 원천 대조)
6. **E Meta 광고**: start_date < 2026-04-01 광고 카드가 사라졌는가
7. **D BSR 탭**: 시계열이 4/1 이후 구간만 그려지는가 (이 케이스 BSR 원천은 5/2까지만 있음 → 4/1~5/2 구간만 나오는 게 정상)
8. **비반영 축 라벨**: 매출 표·SKU 수치가 그대로(4/2~5/1 스냅샷)이고 배너가 "매출 30d 스냅샷은 전 기간 기준"을 고지하는가
9. **해제**: "전체 기간" 클릭 → 전 수치가 기존 전 기간 값으로 원복되는가 (KPI 7,858 등)
10. **부작용**: 다른 케이스(기간 미설정)가 영향 없는가 1건 스팟 (예: Nature Republic) + 콘솔 에러 0
11. **성능**: 기간 적용 렌더가 전 기간 대비 비슷하거나 빠른가 (체감 + Network 탭)
