## 4. partial 상태 (interpret-tag 전용)

배치 열거 완료 후에도 미태깅 잔여(adRemaining>0 || videoFailed>0)면 `phase_runs.status='partial'` + stats에 잔여건수. **재실행(force 불필요 — null인 것만 대상)하면 이어서 진행, 재과금 없음.**
